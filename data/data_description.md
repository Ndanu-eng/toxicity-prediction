# Dataset Description

## Overview
This dataset contains molecular descriptors for 171 chemical compounds, with the goal of predicting toxicity (Toxic vs NonToxic).

## Target Variable
- **Class**: Binary classification
  - `Toxic`: 56 samples (32.7%)
  - `NonToxic`: 115 samples (67.3%)

## Features
- **Total features**: 1203 molecular descriptors
- **Types**: Constitutional, topological, geometrical, electronic, physicochemical
